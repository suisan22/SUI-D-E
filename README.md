# SUI-DEFENSE-ELIMINATE // AI SCANNER

SUI-ELIMINATE is a real-time, AI-powered security suite designed to protect users from malicious web threats, phishing, and unauthorized content streaming. 

## Core Features
1. **SUI Defense (System Shell):** A secure Electron-based shell that monitors system processes and manages hardware permissions to prevent unauthorized screen recording (e.g., OBS).
2. **Token-Based Authentication:** Implements secure digital handshakes to maintain session integrity between the frontend and core system processes.
3. **AI URL Scanner:** Uses Google Gemini to perform deep, lexical, and contextual analysis on URLs, proactively identifying phishing attempts and imposter branding.
4. **Web Detector (Image Forensics):** Analyzes visual content to identify unique versus reused digital media, aiding in forensic threat detection.

## Tech Stack
* **Frontend:** React, Electron
* **Backend:** FastAPI, Python
* **AI Engine:** Google Gemini, Imagga API
* **Database:** Firebase Realtime Database
